Использование миксина b_

				LeColobok = React.createClass({
				mixins: [ BEM ],
				getInitialState() { return { isGood: true }; },
				render() { return <button 
					className={this.b_('_isGood')}>Cancel</button>
			  		// 'leColobok leColobok_isGood'
			  }})

Еще примеры для b_

  1. b_([ '_isTasty', '_escapedFromGranpa' ]) ->-> // this.state === {escapedFromGranpa : false} 'leColobok leColobok_isTasty'
  2. b_('-smile_xxl') ->-> 'leColobok-smile leColobok-smile_xxl'
